## 管理端 v0.1.4

### 新增

- **CloudBase COS 自动更新**：release 版启动时检查 `admin-updates/latest.json`，有新版本弹出系统更新对话框；打 tag 后 CI 自动上传安装包与清单到云存储 CDN
- **首页接口增强**：`portfolioHome` 返回项目轮播图（`images`）与展示价（`price`）
- **首页配置扩展**：支持顶部轮播模式（视频/图片）、轮播图与切换间隔（schema 与共享类型已对齐）

### 修复 / 改进（自 v0.1.3 起）

- macOS 安装包嵌入 ffmpeg 动态库，修复未安装 Homebrew 的机器上双击闪退（v0.1.3）

### 安装包

| 平台 | 产物 |
|------|------|
| macOS Apple Silicon | `.dmg`、`.app.tar.gz` |
| macOS Intel | `.dmg`、`.app.tar.gz` |
| Windows x64 | NSIS `.exe`、`.msi` |

### 自动更新说明

- 更新源：`https://cloud1-d7gyfke5pcc4a88a5.tcb.qcloud.la/admin-updates/latest.json`
- **v0.1.3 及更早版本**不含更新器，需手动安装本版；**v0.1.4 起**可收到后续版本推送
- 云存储需将 `admin-updates/` 设为公有读；CI 需配置 Secret `TAURI_SIGNING_PRIVATE_KEY`

### 升级注意

- 无 Apple / Windows 代码签名证书时，安装或更新时系统仍可能提示「无法验证开发者」，内部分发可忽略
- macOS 打包请使用 `pnpm tauri:build`（含 ffmpeg dylib 嵌入步骤）
