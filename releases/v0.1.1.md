## 管理端 v0.1.1

历史清理后重新发版的基线版本。

### 主要能力

- **桌面管理端**：Tauri 2 + React，直连 CloudBase 管理四级内容（分类 / 项目 / 素材卡片 / 素材详情）与首页设置
- **macOS ffmpeg 自包含**：安装包嵌入动态库，未安装 Homebrew 的机器也可运行
- **CloudBase COS 自动更新**：release 版启动检查 `admin-updates/latest.json`，有新版本弹出系统更新对话框
- **CI 三平台打包**：打 tag 后 GitHub Actions 构建 macOS ARM / macOS Intel / Windows x64 安装包

### 安装包

| 平台 | 产物 |
|------|------|
| macOS Apple Silicon | `.dmg`、`.app.tar.gz` |
| macOS Intel | `.dmg`、`.app.tar.gz` |
| Windows x64 | NSIS `.exe`、`.msi` |

### 自动更新

- 更新源见 `admin/README.md` 与 `tauri.conf.json` 中的 endpoint
- CI 需配置 Secret `TAURI_SIGNING_PRIVATE_KEY`；环境变量通过 Repository secrets 注入，不进 Git
