# 云开发（CloudBase）

本目录用于存放微信小程序云开发侧资源（云函数等），对齐 Taro 文档 `wxcloudbase` 模板结构。

## 目录结构

- `cloud/functions`: 云函数目录（每个函数一个子目录）

## 快速开始（微信开发者工具）

1. 打开微信开发者工具，确保当前项目 **不是游客模式**，并且已开通云开发。
2. 在云开发面板中选择正确环境（例如 `cloud1-xxxx`），并与 `.env` 中 `TARO_APP_CLOUD_ENV_ID` 保持一致。
3. 右键 `cloud/functions/<函数名>` 目录，选择 **上传并部署：云端安装依赖**。

> 注意：本项目的前端调用函数名为 `portfolioHome`，需要确保云端已部署同名函数。
